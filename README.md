# 🎓 Student Management System (Flask)

## 📌 Project Description

This is a simple **Student Management System** built using Flask.
It allows users to manage student records with full CRUD operations, search functionality, pagination, and user authentication.

---

## 🚀 Features

* ➕ Add Student
* 📋 View Students List
* ✏️ Edit Student Details
* ❌ Delete Student
* 🔍 Search Students (by name & course)
* 📄 Pagination (handle large data)
* 🎨 Clean UI using Bootstrap

---

## 🛠️ Technologies Used

* Python (Flask)
* SQLite (Database)
* HTML, CSS (Bootstrap)
* SQLAlchemy (ORM)

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/student-management.git
```

2. Navigate to project folder:

```
cd student-management
```

3. Install dependencies:

```
pip install flask flask_sqlalchemy
```

4. Run the application:

```
python app.py
```

5. Open browser:

```
http://127.0.0.1:5000/
```
